About the Campus Mapping Service


Why CAMPS?

Ever been lost on the UW campus? Ever wondered if there’s a better way to get to your next class?

Well wonder no more.

CAMPS (Campus Mapping Service) is a web service that allows you to easily plan out your whole day’s walking (or running, biking, wheelchair) schedule, from building to building. Simply type in building names in the order on your schedule. CAMPS will display a map of the shortest-route path from building to building, as well as time and distance estimates.

  •   Never get lost again.
  •   Always go the shortest route.
  •   Plan your trip precisely so you can grab those last oh-so-needed minutes of sleep in the morning.

Don’t travel a campus without it.

The People

Written by students, for students. The UW Campus Mapping Service was a project for an undergraduate-level Computer Science class (CSE 403) at the University of Washington during Winter quarter, 2007. The application was developed in its entirety from scratch over a period of ~6 weeks by a team of 6 incredibly handsome Computer Science students:

  • Ryan Adams - Client/Server Integration, Server Code, and Standalone Application
  • Jian Chen - Server Code and Unit Testing
  • Andrew Hitchcock - Client/Server Integration, (AJAX) Building Lookup Functionality, and Destination List
  • Justin Lundberg - Search Algorithm and Server Code
  • Dale Olson - Scrolling (AJAX) Map Interface and Associated Controls
  • Charley Zhang - Tomcat Development/Integration, Server Code, and Documentation

Cool Technology

  • The graphical (AJAX) portion of the application was developed with help from the Google Web Toolkit (and is written in Java, believe it or not!)
  • We also developed a powerful utility to allow easy modification of map destinations and paths.
  • The server (which does the complicated algorithmic work for CAMPS) was also written in Java and runs on Apache Tomcat.
  • For calculating shortest paths quickly we used Dijkstra's algorithm, which you get to learn about in the CS Data Structures class.
